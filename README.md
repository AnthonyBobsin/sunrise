## Requirements
* node `^4.5.0`
* yarn `^0.17.0`

## Initial Setup
1. `git clone https://github.com/bsheikh/sunrise.git`
2. `cd sunrise`
3. `yarn install`
4. `yarn run dev:sandbox`
5. Verify at `localhost:4000`

## Browser Support
* Chrome
* Mozilla
* Safari
